# Button Variants

```bash
primary -  Blue

secondary - Gray

danger - red

light - white

default - gray

```

# Button Sizing

```bash
small - px-4 py-2 (padding l-r) (padding top-bottom)

medium - px-6 py-2

large - px-8 py-2

## you need to wrap in div to customize it lenght and width
custom - w-full h-full
```

```bash
sample syntax:
<Button size='large' variant="primary" > Sign Up </Button>
```
